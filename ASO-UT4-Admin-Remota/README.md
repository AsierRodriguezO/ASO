# README

Este documento describe la configuración y el uso de Windows Admin Center (WAC) en un entorno de infraestructura que incluye servidores Windows y Ubuntu. 

## Contenido del Archivo WAC.md

- **Infraestructura**: Se detallan los servidores utilizados, incluyendo Windows Server 2025 y Ubuntu Server 24.04, así como el equipo administrador que opera con Windows 11.
- **Conexiones**: Se menciona que todas las conexiones se realizan a través del navegador en Windows 11.
- **Instalación de WAC**: Se proporciona información sobre la herramienta Windows Admin Center, el protocolo HTTPS y el puerto configurado en el sistema, junto con imágenes que ilustran el proceso de instalación.

Este README proporciona una visión general de los elementos clave abordados en el archivo WAC.md, facilitando la comprensión de la infraestructura y la herramienta utilizada.
